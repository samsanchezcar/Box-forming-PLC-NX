# MATLAB

Contenido asociado al diseño y validación de perfiles de movimiento:

- `original/`: archivos originales `.mlx`.
- `extracted/`: versiones recuperadas en `.m` para lectura rápida y versionamiento.
- `output/`: resúmenes de salida generados por los Live Scripts.

Archivos principales:

- `profile_design_extracted.m`: diseña perfiles trapezoidales por etapas para Y, RotX y RotZ.
- `optimizar_perfil_8cpm_extracted.m`: búsqueda en malla con refinamiento para 8 cajas/min.
- `perfil_trapezoidal_sliders.m`: interfaz interactiva con sliders para explorar aceleraciones, reposos y métricas.
